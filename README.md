## IMG To Text Converter 
You just need to place your images in the images_folder and run the python file it will generate text of each image and save it inside extracted texts folder inside images folder <br><br>
To run the code:<br>
1. Run this command to install the dependendies

```
pip install -r requirements.txt
```
2. After this, run this command inside the same directory to run the actual code:

```
python main.py
```
